Image Management

Build an image: docker build -t <image_name> .

List images: docker images

Remove an image: docker rmi <image_name>

Pull an image from Docker Hub: docker pull <image_name>

Push an image to Docker Hub: docker push <username>/<image_name>

Prune unused images: docker image prune

Container Management

Run a container: docker run <image_name>

Run a container with a custom name: docker run --name <container_name> <image_name>

Run a container in detached mode: docker run -d <image_name>

Map ports: docker run -p <host_port>:<container_port> <image_name>

Start a container: docker start <container_name>

Stop a container: docker stop <container_name>

Remove a container: docker rm <container_name>

List running containers: docker ps

List all containers (including stopped): docker ps -a

Inspecting and Debugging

Inspect a container: docker inspect <container_name>

View logs of a container: docker logs -f <container_name>

Check resource usage: docker container stats

View running processes in a container: docker container top <container_name>

Interacting with Containers

Execute a command in a running container: docker exec -it <container_name> <command>

Open a shell inside a container: docker exec -it <container_name> sh

System Cleanup

Remove unused containers, images, and networks: docker system prune

Remove all unused images: docker image prune -a

Docker Compose

Start services: docker-compose up

Stop services: docker-compose down

View running services: docker-compose ps
