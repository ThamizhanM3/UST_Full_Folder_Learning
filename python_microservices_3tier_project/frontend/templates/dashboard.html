📝 Quick Overview

This application demonstrates a microservices architecture with separate services for:

  • User Service: Manages user accounts and profiles
  • Product Service: Handles product catalog and inventory
  • Order Service: Processes and tracks customer orders
  • Payment Service: Handles payment processing
  • Shipping Service: Manages order shipping and logistics
  • Notification Service: Sends notifications and alerts
  • Inventory Service: Tracks stock levels

Use the navigation menu above to manage users, products, and orders across different services.
